## Prerequisites

- [Astro](https://astro.build) 6
- Node.js 22.12.0 or later
- A [bunny.net](https://bunny.net) account

## Configuration

Add the adapter to your `astro.config.mjs`:

```js
// astro.config.mjs
import { defineConfig } from "astro/config";
import bunny from "astro-bunny-adapter";

export default defineConfig({
  output: "server",
  adapter: bunny(),
});
```

:::caution
`output: "server"` is required. Static output is not supported because
bunny.net Edge Scripting is designed for on-demand server rendering.
:::

## Build output

Running `astro build` produces two directories:

```
dist/
├── client/   # Static assets (CSS, JS, images, prerendered pages)
└── server/
    └── entry.mjs   # Server entrypoint for Edge Scripting
```

- **`dist/client/`** — Upload these files to a [Bunny Storage Zone](https://docs.bunny.net/docs/storage-overview)
  connected to your Pull Zone.
- **`dist/server/entry.mjs`** — Deploy this file as your Edge Script.

See [Deploying](/deploying) for step-by-step instructions.
